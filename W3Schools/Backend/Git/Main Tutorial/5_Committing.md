
# GIT: W3Schools - Main Tutorial
## Section 5: Committing

Each commit is essentially a save point in the history of the project.

### Basic Git Commands:
| Command | Action |
| git commit -m <message> | creates a commit with message |

| Status Flag | Meaning |
| ?? | Untracked files |
| A | Files added to stage |
| M | Modified files |
| D | Deleted files |
